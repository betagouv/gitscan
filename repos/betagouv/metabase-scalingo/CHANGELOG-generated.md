## Changelog : metabase-scalingo (30 derniers jours)

### Résumé
Ce changelog présente une mise à jour technique pour assurer la compatibilité de Metabase avec les dernières versions de Java fournies par Scalingo. Cette mise à jour ne devrait pas impacter directement l'utilisation de Metabase par les utilisateurs finaux, mais elle garantit la stabilité et la pérennité du service.

### Évolutions techniques
- Mise à jour vers JDK 25 pour assurer la compatibilité avec l'infrastructure Scalingo. [#7](https://github.com/betagouv/metabase-scalingo/issues/7)
