## Changelog : oauth2-proxy-buildpack (30 derniers jours)

### Résumé
Ce buildpack a été mis à jour pour permettre une authentification plus fine via OAuth2. Il est désormais possible de spécifier une liste d'adresses email autorisées à accéder à l'application, renforçant ainsi la sécurité. De plus, le projet a été renommé et la documentation a été mise à jour pour refléter ce changement.

### Évolutions fonctionnelles
- Ajout de la possibilité de définir une liste blanche d'adresses email autorisées pour l'authentification via OAuth2. Ceci permet de restreindre l'accès à l'application à des utilisateurs spécifiques.  [#1](https://github.com/betagouv/oauth2-proxy-buildpack/issues/1)

### Évolutions techniques
- Renommage du projet et mise à jour du fichier README pour refléter le nouveau nom. [#2](https://github.com/betagouv/oauth2-proxy-buildpack/issues/2)
