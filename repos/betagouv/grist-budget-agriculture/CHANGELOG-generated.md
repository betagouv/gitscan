## Changelog : grist-budget-agriculture (30 derniers jours)

### Résumé
Ce mois-ci, les améliorations se concentrent sur la simplification du traitement des données budgétaires, la correction de problèmes d'exécution planifiée et la suppression de code obsolète lié à un module expérimental. La documentation a également été mise à jour et un paquet Python a été créé.

### Évolutions techniques
- Simplification du traitement des INFBUD MTE (#f524729)
- Suppression du module PPA et renforcement du lint automatique (#e1ff3bd)
- Suppression du script exploratoire de la PPA / Peste porcine africaine (#cf041f8)
- Correction des cron jobs (#c48281d)
- Correction du chemin vers l'application frontend (#41bca99)
- Création d'un paquet Python (#71899a6)

### Autres changements
- Mise à jour de la documentation (#5c62155)
