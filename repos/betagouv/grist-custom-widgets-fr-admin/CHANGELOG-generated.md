## Changelog : grist-custom-widgets-fr-admin (30 derniers jours)

### Résumé
Ce changelog présente les récentes améliorations apportées aux widgets personnalisés pour Grist, destinés à l'administration française. Les modifications incluent une correction concernant le widget de géocodage et une mise à jour de la documentation pour refléter le lien API correct.

### Évolutions fonctionnelles
- Correction d'un problème dans le widget de géocodage (#167).

### Évolutions techniques
- Mise à jour de Next.js de la version 16.0.10 à la version 16.1.5 (#164).
- Modification du lien API dans le README du widget de géocodage pour assurer son bon fonctionnement.
