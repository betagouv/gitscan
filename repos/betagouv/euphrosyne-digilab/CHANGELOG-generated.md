## Changelog : euphrosyne-digilab (30 derniers jours)

### Résumé
Ce changelog fait état de plusieurs mises à jour techniques visant à maintenir la plateforme Euphrosyne Digilab à jour avec les dernières versions de ses dépendances. Une amélioration de la qualité du code a également été effectuée grâce à l'application automatique du formateur de code Prettier.

### Évolutions techniques
- Mise à jour de Next.js de la version 15.5.9 à la version 16.1.1 (#283)
- Mise à jour de Webpack de la version 5.102.1 à la version 5.105.0
- Mise à jour de React DOM de la version 19.2.3 à la version 19.2.4
- Mise à jour de TypeScript de la version 24.0.10 à la version 25.0.10
- Refactorisation du code pour appliquer le formateur Prettier, améliorant la cohérence du style de code. (#286)
- Mise à jour de Prettier de la version 3.5.3 à la version 3.8.0
- Mise à jour de tss-react de la version 4.9.18 à la version 4.9.20 (#285)
