## Changelog : beta.gouv.fr (30 derniers jours)

### Résumé
Ce changelog présente les récentes mises à jour de beta.gouv.fr, la plateforme de l'Incubateur de Services Numériques de l'État. Les évolutions concernent principalement l'ajout de documentation API, l'amélioration des données sur les technologies utilisées par les startups, et des mises à jour régulières de la base de données.

### Évolutions fonctionnelles
- Ajout d'une documentation Swagger pour l'API, facilitant son utilisation et sa compréhension. (#21450)
- La liste des startups affichées sur la plateforme exclut désormais les incubateurs "sgmas" hérités. (#21427)
- Ajout de l'information sur les technologies utilisées par les startups dans l'API. (#21434)
- Amélioration de la propriété `incubator.hidden` pour une meilleure gestion de la visibilité des startups. (#21444)

### Évolutions techniques
- Mises à jour régulières de la base de données (plusieurs commits référencés : #21458, #21457, #21456, #21455, #21451, #21449, #21448, #21445, #21442, #21441, #21440, #21438, #21436, #21435, #21432, #21431, #21430, #21428, #21426).

### Autres changements
- Ajout de documentation générale sur l'API. (#21433)
- Bilan du schéma pluriannuel d'accessibilité 2025 ajouté. (#21404)
