## Changelog : beta.gouv.fr (30 derniers jours)

### Résumé
Les dernières mises à jour de beta.gouv.fr se concentrent principalement sur l'amélioration de l'API et de la documentation, ainsi que sur des modifications régulières de la base de données. Une nouvelle fonctionnalité permet d'accéder aux technologies utilisées par les startups.

### Évolutions fonctionnelles
- Ajout d'une documentation Swagger HTML pour l'API (#21450).
- Ajout d'une propriété `incubator.hidden` pour masquer des startups si nécessaire (#21444).
- L'API expose désormais les technologies utilisées par les startups via le point d'accès `startups.technos` (#21434).

### Évolutions techniques
- Amélioration de la documentation de l'API (#21433).

### Autres changements
- Plusieurs modifications de la base de données ont été effectuées (#21458, #21457, #21456, #21455, #21451, #21449, #21448, #21445, #21442, #21441, #21440, #21438, #21436, #21435, #21432, #21431, #21430, #21428). Ces changements sont gérés par le bot `secretariat-bot`.
