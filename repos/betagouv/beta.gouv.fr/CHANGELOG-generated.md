## Changelog : beta.gouv.fr (30 derniers jours)

### Résumé
Au cours des dernières semaines, le site beta.gouv.fr a bénéficié d'améliorations continues de sa base de données, ainsi que de l'ajout de nouvelles fonctionnalités à l'API, notamment concernant les informations sur les technologies utilisées par les startups. Une documentation API a également été ajoutée pour faciliter l'utilisation des services.

### Évolutions fonctionnelles
- Ajout d'une documentation Swagger HTML pour l'API (#21450)
- Ajout d'une propriété `incubator.hidden` pour gérer la visibilité des startups (#21444)
- L'API expose désormais les technologies utilisées par les startups via le point d'accès `startups.technos` (#21434)

### Évolutions techniques
