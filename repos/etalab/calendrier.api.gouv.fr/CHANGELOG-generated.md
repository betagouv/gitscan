## Changelog : calendrier.api.gouv.fr (30 derniers jours)

### Résumé
Ce mois-ci, l'API des jours fériés a bénéficié d'une mise à jour de la documentation avec des exemples concrets pour faciliter son utilisation. Un job de maintenance a également été mis en place pour assurer la disponibilité continue de l'API.

### Évolutions fonctionnelles
- Ajout d'exemples concrets dans la documentation OpenAPI pour illustrer l'utilisation de l'API. [#27](https://github.com/etalab/calendrier.api.gouv.fr/issues/27)

### Évolutions techniques
- Mise en place d'un job GitHub Actions pour maintenir l'API active et garantir sa disponibilité.
