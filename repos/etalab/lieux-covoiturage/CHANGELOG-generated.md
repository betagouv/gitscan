## Changelog : lieux-covoiturage (30 derniers jours)

### Résumé
Ce changelog présente les récentes mises à jour du schéma de données pour les lieux de covoiturage. Les changements incluent l'ajout d'une nouvelle valeur possible pour le type de lieu et une mise à jour de la configuration du workflow de test pour améliorer la gestion du cache.

### Évolutions fonctionnelles
- Ajout d'une nouvelle valeur pour l'énumération du type de lieu (#31). Cela permet de mieux catégoriser les différents types de lieux de covoiturage.

### Évolutions techniques
- Mise à jour du workflow de test (`test.yml`) pour améliorer la gestion du cache (#30). Cette modification vise à optimiser les performances des tests et à garantir une meilleure fiabilité.
