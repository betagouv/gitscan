## Changelog : domifa (30 derniers jours)

### Résumé
Ce changelog couvre les dernières améliorations apportées à DomiFa, la plateforme de gestion de la domiciliation. Les changements incluent une correction concernant le suivi Matomo, des améliorations de la synchronisation avec Brevo, des corrections d'affichage et de gestion des logs dans l'interface administrateur, ainsi qu'une migration vers le gestionnaire de paquets pnpm.

### Évolutions fonctionnelles
- Correction d'un bug affectant le suivi Matomo, améliorant ainsi la collecte de données analytiques. (#4069)
- Correction de l'affichage de la date de dernière connexion dans l'interface administrateur.
- Amélioration de la synchronisation avec le service Brevo.

### Évolutions techniques
- Migration du projet vers le gestionnaire de paquets pnpm, pour une meilleure gestion des dépendances et des performances.
- Suppression de configurations inutilisées et de logs obsolètes dans le backend.
- Correction de problèmes liés aux tâches cron.
- Amélioration des tests unitaires backend.

### Autres changements
- Nettoyage et refactorisation du code dans le cadre de la migration pnpm.
