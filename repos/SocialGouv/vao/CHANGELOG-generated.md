## Changelog : vao (30 derniers jours)

### Résumé
Ce changelog présente les évolutions récentes de l'application vao, dédiée à la dématérialisation des procédures liées aux séjours pour personnes handicapées. Les dernières mises à jour incluent des améliorations de l'interface utilisateur, des corrections de bugs impactant l'utilisation de certaines fonctionnalités, l'intégration de *feature flags* pour une gestion plus flexible des fonctionnalités, et des avancées dans l'intégration avec le stockage S3.

### Évolutions fonctionnelles
- Correction d'un bug empêchant la désactivation d'un hébergement [#1204](https://github.com/SocialGouv/vao/issues/1204).
- Correction d'un problème d'affichage des utilisateurs OVA secondaires [#1197](https://github.com/SocialGouv/vao/issues/1197).
- Correction d'un bug dans le filtre de liste des DS pour une organisation [#1196](https://github.com/SocialGouv/vao/issues/1196).
- Implémentation de la page d'authentification à deux facteurs (2FA) [#1183](https://github.com/SocialGouv/vao/issues/1183).
- Ajout de boutons pour améliorer l'accessibilité des tableaux (conformité RGAA) [#1186](https://github.com/SocialGouv/vao/issues/1186).
- Ajout d'une bordure à la page 2FA pour une meilleure visibilité [#1215](https://github.com/SocialGouv/vao/issues/1215).

### Évolutions techniques
- Intégration de *feature flags* pour une gestion plus souple des fonctionnalités [#1205](https://github.com/SocialGouv/vao/issues/1205).
- Avancement de l'intégration avec le service de stockage S3 [#1198](https://github.com/SocialGouv/vao/issues/1198).
- Amélioration du workflow SonarQube pour une meilleure analyse de la qualité du code [#1202](https://github.com/SocialGouv/vao/issues/1202).
- Optimisation du middleware de vérification des permissions pour les hébergements (correction de requêtes SQL dupliquées et gestion des erreurs) [#1204](https://github.com/SocialGouv/vao/issues/1204), [#1197](https://github.com/SocialGouv/vao/issues/1197).

### Autres changements
- Mise à jour du template de pull request pour améliorer la qualité des contributions [#1191](https://github.com/SocialGouv/vao/issues/1191).
- Publication des versions 1.26.0 [#1192](https://github.com/SocialGouv/vao/issues/1192) et 1.26.1 [#1200](https://github.com/SocialGouv/vao/issues/1200).
