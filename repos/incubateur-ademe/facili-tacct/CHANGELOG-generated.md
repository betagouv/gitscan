## Changelog : facili-tacct (30 derniers jours)

### Résumé
Les dernières mises à jour de Facili-tacct se concentrent sur l'amélioration de l'expérience utilisateur, notamment dans la section "patch4" concernant les aléas et les recettes. Des corrections de bugs et des ajustements d'interface ont été apportés pour une meilleure clarté et fonctionnalité. Des travaux de préparation pour une prochaine étape de déploiement (MEP) ont également été réalisés.

### Évolutions fonctionnelles
- Amélioration de l'affichage et de la gestion des aléas dans la section "patch4" (#1054).
- Correction de l'affichage des données statistiques pour les types de cultures.
- Ajout d'une condition d'affichage pour l'intensité modérée dans "patch4".
- Mise à jour du texte et des formulations dans "patch4" pour une meilleure compréhension.
- Ajout d'un bouton d'export pour déclencher une enquête.
- Correction de l'affichage des surfaces agricoles secrètes.
- Correction du problème d'affichage des aléas dans un iframe.
- Affichage de tous les aléas pour un niveau d'aggravation modéré.

### Évolutions techniques
- Génération des modèles Prisma corrigée et optimisée.
- Refactoring du code pour utiliser `toSorted` pour le tri.
- Suppression de logs de console inutiles avant déploiement.
- Préparation pour le déploiement MEP du 16/02/2026.
- Intégration des tuiles de carte pour le débroussaillement.
- Intégration du menu latéral pour le débroussaillement.

### Autres changements
- Mise à jour du budget.
- Amélioration du titre des articles.
- Ajout d'une notice sur la page d'accueil.
- Correction de l'affichage des recettes dans "patch4".
- Suppression du niveau d'aggravation dans la partie statique de l'interface.
- Ajout d'une notice sur la page d'accueil.
