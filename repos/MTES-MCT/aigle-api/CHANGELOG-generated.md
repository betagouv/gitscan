## Changelog : aigle-api (30 derniers jours)

### Résumé
Cette mise à jour se concentre sur l'amélioration de l'importation de données depuis la source Lucca. Les utilisateurs pourront bénéficier d'un processus d'importation plus fiable et potentiellement plus rapide.

### Évolutions fonctionnelles
- Amélioration de la commande `import_from_lucca` pour une meilleure gestion de l'importation des données Lucca. (#52)

### Évolutions techniques
- Fusion de la branche `develop` dans la branche principale.
