## Changelog : ecobalyse-data (30 derniers jours)

### Résumé
Cette mise à jour apporte des améliorations significatives aux données et aux fonctionnalités d'Ecobalyse. De nouveaux ingrédients avec des métadonnées prédictives ont été ajoutés, ainsi que des compléments pour les forêts et des ajustements concernant les pâturages permanents. Des optimisations ont été apportées pour améliorer la performance lors de l'exportation de toutes les données.

### Évolutions fonctionnelles
- Ajout de nouveaux ingrédients avec des métadonnées prédictives (#196)
- Ajout de compléments pour les forêts (#208)
- Correction concernant les pâturages permanents (#213)
- Amélioration de la performance lors de l'exportation de toutes les données, évitant de ralentir le système (#212)
- Ajout de la masse par unité pour certains processus (#204)

### Évolutions techniques
- Refactorisation : Ajout d'une vérification pour détecter les activités dupliquées (#205)
- Amélioration de l'ordre des activités dans le fichier `activities.json` (#216)
- Modification de l'ordre d'affichage du "Proxy" dans le nom d'affichage (#202)
